# 🚀 Quick Start

## Run RideWise in 3 Steps

### Step 1: Open Terminal

Navigate to the project folder:

```bash
cd ridewise-app
```

---

### Step 2: Install Dependencies (First Time Only)

```bash
npm install
```

Wait ~1 minute for packages to install.

---

### Step 3: Start the App

```bash
npm run dev
```

You'll see:

```
➜  Local:   http://localhost:5173/
```

---

### Step 4: Open in Browser

Open your browser and go to:

**http://localhost:5173**

---

## ✅ You're Done!

The app is now running. You should see:

1. **RideWise header** at the top
2. **"Compare rides. Book smarter."** headline
3. **Two input fields:** Pickup and Destination
4. **"Compare Rides" button**

---

## 🛑 Stop the App

Press `Ctrl+C` in the terminal where it's running.

---

## 📱 Test on Mobile

1. Keep the dev server running
2. Resize your browser window to mobile width
3. See the responsive design in action

## 🔥 Hot Tips

- **Changes auto-reload:** Edit any file → browser refreshes automatically
- **Multiple devices:** Find Network URL in terminal output (use --host flag)
- **Debug mode:** Open browser DevTools (F12) to see console logs

---

## ❓ Having Issues?

**Port already in use?**
```bash
npm run dev -- --port 3000
```

**Blank screen?**
1. Check browser console (F12)
2. Refresh page (Ctrl+R)
3. Clear browser cache

**npm not found?**
Install Node.js: https://nodejs.org
